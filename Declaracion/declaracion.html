Declaración de Accesibilidad de MediUA

MediUA se ha comprometido a hacer accesible su sitio web, de conformidad con el Real Decreto 1112/2018, de 7 de septiembre, sobre accesibilidad de los sitios web y aplicaciones para dispositivos móviles del sector público.

Esta es una declaración de accesibilidad de MediUA.

Estado de conformidad

Las WCAG (Pautas de Accesibilidad para el Contenido Web) definen requisitos para que los diseñadores y desarrolladores mejoren la accesibilidad para las personas con discapacidad. Define tres niveles de conformidad: Nivel A, Nivel AA y Nivel AAA. MediUA es parcialmente conforme con WCAG 2.2 nivel A. Parcialmente conforme significa que algunas partes del contenido no cumplen completamente con el estándar de accesibilidad.

Consideraciones adicionales de accesibilidad

Aunque nuestro objetivo es cumplir con el nivel A de conformidad de las WCAG 2.2, también hemos aplicado algunos Criterios de Éxito del nivel AA: Orientación, ningún evento/actividad depende de la orientación. Identificar el objetivo de entrada, no hay campos de entrada ambiguos. Contraste (mínimo), todas las páginas cumplen el mínimo contraste. Cambiar el tamaño de texto, no se pierde la estructura con el aumento del tamaño al 200%. Imágenes de texto, no se usa texto dentro de las imágenes. En todas las páginas se usa texto real al alcance de lectores de pantalla. Reflujo, no se pierde la estructura con el aumento del tamaño al 400%. Contraste no textual, se mantiene el contraste necesario en botones, gráficos. Varias maneras, se puede acceder al mismo contenido mediante varias opciones (menú navegación o enlaces entre páginas). Encabezados y etiquetas, todos los títulos y etiquetas describen el propósito de su contenido sin ambigüedad. Foco visible, el usuario sabe en todo momento dónde se encuentra el foco al navegar por teclado, ya sea mediante un borde negro o la barra vertical parpadeando en los campos. Foco no tapado (mínimo), no se oculta el foco en ninguna página. Idioma de las partes, cuando ocurre un cambio de idioma está indicado. Navegación consistente, todas las páginas tienen la misma estructura de navegación.

Y algunos Criterios de Éxito del nivel AAA: Identificar el objetivo, se usan landmarks para identificar regiones. Contraste (mejorado), todas las páginas cumplen el contraste mejorado. Teclado (sin excepción), funcionalidad completa mediante teclado. Sin límite de tiempo, ningún evento/actividad presenta límite de tiempo. Ubicación y propósito del enlace (solo enlace), en todo momento el usuario es capaz de localizar en qué parte de la página web se encuentra gracias a las migas de pan y el mapa del sitio web situados en todas las páginas. Encabezados de sección, cada sección tiene un título claro con jerarquía definida. Foco no tapado (mejorado), no se oculta el foco en ninguna página. Mecanismos de entrada simultánea, no se restringe el modo de entrada. Abreviaturas, todas las abreviaturas están definidas con el atributo title de la etiqueta <abbr>. Nivel de lectura, debido a la naturaleza de la página web, la estructura de las frases es comprensible, pero el vocabulario médico empleado requiere un conocimiento previo. Cambios de contexto pedidos, la información no se actualiza sola en ninguna página.
